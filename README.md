# Arbeidskrav 1 Js

## Teknologi
- JavaScript, html, css
- Node.js

## Pakker
- live-server
- jest
- jest-environment-jsdom



## Hvordan installere, kjøre og teste
- Node.js må være installert på maskinen
- I terminalen, naviger til projekt mappen
- npm install
- npm run
- npm test
